Escrevendo no blog com markdown

Para escrever novos posts no blog, devemos escrever com a sintaxe do markdown,
arquivos com extensão .md.
O editor de textos Atom é uma excelente ferramenta para escrevermos em markdown com um mínimo de esforço, especialmente se ele tem os plugins markdown-preview e markdown-writer.

Veja também essa outra página Notas sobre como criar blockquotes em Markdown.
