Exemplo rápido em markdown

Um header h1

Parágrafos são separados por uma linha em branco.

2° parágrafo. Itálico, negrito e monospace.

Listas com ítens:

  • um ítem
  • outro ítem
  • ainda outro ítem

(note que a lista é indentada 4 caracteres para a direita)

Citações em bloco são escritas assim.

Elas podem ocupar vários parágrafos, se desejado.

Um header h2

Uma lista numerada:

  1. primeiro ítem
  2. segundo ítem
  3. terceiro ítem

(note que a lista novamente é indentada 4 caracteres para a direita)

Escrevendo código:

# uma iteração com 'for' ...
for i in 1 .. 10 { processa_algo(i) }

Agora um exemplo de lista dentro de outra (aninhada):

  1. Primeiro, obtenha os ingredientes:

    • cenouras
    • aipo
    • lentilhas

  2. Ferva água.

  3. Jogue tudo num recipiente e siga o algorítmo:

    procure uma colher de pau
descubra o recipiente
mexa
cubra o recipiente
balance a colher (precariamente)
aguarde 10 minutos
vá para o primeiro passo (ou desligue o fogo quando pronto)

    Não bata a colher, ou ela cairá.
